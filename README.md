## How to Run

1. Install dependencies
   npm install

2. Type-check the project (no errors should be reported)
   npx tsc --noEmit

3. Compile TypeScript to JavaScript
   npx tsc
   Output will be generated in the dist/ folder.

4. Run the compiled code
   node dist/index.js

## Requirements

- Node.js
- TypeScript (installed as a dev dependency via npm install)
